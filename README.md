# carpetasVisualstudio
Descripcion e Instrucciones = Este es un juego desarrollado para la up.e, y se trata de: vos sos el cuadrado celeste mientras que los
enemigos son los cuadrados rojos, tu objetivo es esquivar a tus enemigos, usando "left arrow" para moverte a la izquierda, y "right
arrow" para moverte hacia la derecha.

Historia = Un alien celeste fue capturado por los aliens rojos, celeste logra escapar de los rojos pero hay un problema, los rojos se
enteraron que celeste habia escapado y lo fueron a buscar....Â¿Tu objetivo? ayudar a celeste para que los rojos no lo atrapen.
